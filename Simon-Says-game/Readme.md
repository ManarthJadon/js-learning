# 🎵 Simon Says Game! 🕹️

Welcome to **Simon Says**, an exciting memory game with a twist! This isn't your usual Simon Says where the entire color pattern is repeated. Instead, this game flashes **only the next color**, and you must repeat the entire sequence—from the start—to progress to the next level. 🌈🧠

---

## 🎯 Objective

Memorize and repeat the sequence of colors accurately to move to the next level. The challenge increases with each round as the sequence gets longer! 🔥

---

## 🎮 How to Play

1. 💡 First, press any key to start the game.
2. 🚦 A random button will flash to indicate the first color in the sequence.
3. 🖱️ Click on the buttons in the correct order.
4. ✅ If you repeat the sequence successfully, you move to the next level, and the next random button in the sequence flashes.
5. ❌ If you click the wrong button, the game ends—try again!

---

## 🛠️ Features

- 🎨 Four vibrant buttons: **Red**, **Blue**, **Green**, and **Yellow**.
- 🌈 **Unique Challenge**: Instead of repeating the color sequence, the game flashes the **next color only**, and you must recall the full sequence.
- 🌟 Scoring system that displays your level and score.
- 🤖 Keyboard and mouse-friendly.

---

## 🏃 How to Run

1. Clone the repository: `git clone https://github.com/ManarthJadon/js-learning`
2. Navigate to the `Simon-Says-game` folder.
3. Open `index.html` in your browser.
4. Start enjoying the fun!

---

## 📂 Project Structure

- **`index.html`** — The main HTML file for the game interface.
- **`style.css`** — Adds colors and animations to the buttons.
- **`app.js`** — Includes the game logic.
