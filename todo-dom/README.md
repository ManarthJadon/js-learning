# Todo App (DOM Manipulation) 📝

A dynamic, interactive To-Do List application built with **vanilla JavaScript** and **DOM manipulation**. This project demonstrates real-time task management with a clean, user-friendly interface.

## 🎯 Features

* **✨ Add Tasks**: Dynamically create new todo items from user input
* **🗑️ Delete Tasks**: Remove individual tasks with a single click
* **🎨 Interactive UI**: Real-time DOM updates without page refresh
* **💾 Instant Feedback**: Console logging for debugging and user actions

## 🛠️ Technologies Used

* **HTML5**: Semantic markup for structure
* **CSS3**: Flexbox layout for styling
* **Vanilla JavaScript**: DOM manipulation and event handling
