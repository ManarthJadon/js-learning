# JavaScript Learning Projects

This repository contains fundamental JavaScript programs and mini projects developed while mastering core concepts including variables, loops, functions, arrays, and DOM interactions.

## 📚 Projects Included

### 1. Guess My Favorite Movie 🎬
A simple guessing game showcasing `prompt`, loops, and conditional logic.

**Features:**
- Interactive movie guessing gameplay
- Retry mechanism with quit option
- Reward feedback system

**Tech Stack:** HTML, JavaScript

**How to Run:** Open `guess-my-favorite-movie/index.html` in your browser

---

### 2. Todo App (DOM Manipulation) 📝
A dynamic, interactive To-Do List application built with vanilla JavaScript and DOM manipulation. Demonstrates real-time task management with a clean, user-friendly interface.

**Features:**
- ✨ Add tasks dynamically
- 🗑️ Delete individual tasks
- 🎨 Real-time interactive UI
- 💾 Instant user feedback

**Tech Stack:** HTML5, CSS3, Vanilla JavaScript

**How to Run:** Open `todo-dom/index.html` in your browser

---

## 🎓 Learning Objectives

This repository covers:
- Variables and data types
- Loops and iteration
- Functions and scope
- Arrays and array methods
- DOM manipulation and event handling
- User interaction and validation
